# 環境構築方法
- ```docker-compose up``` でdockerを立ち上げてください
- http://localhost:3001/tasks にアクセスしてWebサイトにアクセス出来ることを確認してください

# 開発環境
- Backend
    - Rails
    - Graphql
- DB
  - PostgreSQL
- Front
  - Next.js
